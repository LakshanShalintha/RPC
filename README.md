Rajarata Pawning Center Website

This is a Next.js project for Rajarata Pawning Center.
It has a splash/loading page, Home, Services, About, and Contact pages.
Navbar and Footer are shared across all pages.

Features

Splash/loading page with 2 seconds delay
Separate pages: Home, Services, About, Contact
Navbar and Footer on all pages
Built with Next.js 13+ and Tailwind CSS

How to Run

Clone the repo:
git clone https://github.com/your-username/pawning-center.git
cd pawning-center

Install dependencies:
npm install

Run the development server:
npm run dev

Open http://localhost:3000
 in your browser
